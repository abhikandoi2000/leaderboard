About Leaderboard

Leaderboard tracks your team's social scores across various services. Here is a list of services it can track currently:

Github
Track user's follower count.
Ask Ubuntu
User's reputation
Facebook
User's subscriber count
Klout
User's score on klout
Last.FM
User's play count(total number of songs scrobbled)
Stackoverflow
User's reputation on the site
Twitter
Follower count on twitter
SPOJ
Number of problems solved on SPOJ
GitScore
User's score on gitscore.com. Generated from your github activity
Hacker News
User's karma on user's profile page
Codechef
Number of problems solved on Codechef
Project Euler
Number of problems solved on project euler
Superuser
User's reputation
